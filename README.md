# It's a Keyboard Game

A fun, interactive web-based keyboard visualizer that brings your keyboard to life with vibrant, seasonal themes!

## 🎮 About

This project displays a virtual keyboard layout on your screen. Each key is randomly assigned a color from the selected theme, creating a visually appealing and dynamic representation of a keyboard. Perfect for keyboard enthusiasts, designers, or anyone who loves colorful interfaces!

## 🌟 Features

- **Interactive Keyboard Layout**: Full QWERTY keyboard representation with all keys including special keys like Esc, Tab, Caps Lock, Shift, Ctrl, etc.
- **Themed Color Schemes**: Choose from multiple seasonal themes:
  - **Spring**: Fresh and vibrant greens and blues
  - **Summer**: Bright and energetic ocean and sunset colors
  - **Autumn**: Warm earth tones and harvest colors
  - **Winter**: Cool icy blues and whites
  - **Custom**: Unique blend of modern colors
- **Dynamic Coloring**: Each key gets a randomly assigned color from the selected theme, ensuring every view is unique
- **Responsive Design**: Clean, modern interface that works on various screen sizes

## 🚀 How to Use

1. Open the [live demo](https://binaryvortex.github.io/Its-a-Keyboard-Game/) in your web browser
2. Select a theme from the dropdown menu
3. Watch as the keyboard keys change colors instantly!
4. Switch themes anytime to see different color combinations

   ```bash

   ```
3. Open `index.html` in your web browser
